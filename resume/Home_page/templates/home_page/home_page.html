О себе

Закончил электротехнический колледж. Работал на заводах, выпускающих военную продукцию.

На данный момент изучаю программирование так как данное направление меня привлекало всю мою жизнь.

Программирование - перспективное направление, которое открывает постоянные пути для развития. Данная профессия дает возможность для карьерного роста.

Навыки и качества

Мои IT навыки:
Знание Я/П Python 3.8.
Умение работы с БД.
Знание фреймоврка Django.
Умение работать с такими иструментами как: IDE, CMD, VENV, HTML, CSS, Bootstrap.

Мои основные технические навыки:
Поиск неисправностей/ремонт/настройка электроники на компонентном уровне.
